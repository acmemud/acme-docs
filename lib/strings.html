        find_nonws(string str, int start)
        Find the first non-space character in a string. Note: functionally equivilent to ArrayLib->anti_search(str, ' ', start), but optimized for speed.
        Parameters:
        str - the string to search
        start - optional start position for search, defaults to 0
        Returns:
        the index of the first non-space character, or strlen(str) if the string only contains spaces

      • match_quote

        match_quote(string str, int start, string quote_chars)
        Find the index of the last character in a possibly quoted string.
        Parameters:
        str - the string to search
        start - optional start position, defaults to 0
        quote_chars - an optional string containing quote characters, defaults to "\"'"
        Returns:
        the index of the last character in the first quoted character sequence, or the index of the last character of the first word

      • unescape

        unescape(string str)
        Unescape all characters in a string by one level. Note: This deals with literal backslashes only. In the examples, and ind the strings this function is meant to deal with "\a" is meant to represent a string containing the two characters '\\' and 'a', not a string containing a single '\a' character.
        Parameters:
        str - the string to unescape
        Returns:
        the unescaped string

      • search_str_unescaped

        search_str_unescaped(string str, int el, int pos, int step)
        Search a string for a character, ignoring escaped characters.
        Parameters:
        str - the string to search
        el - the character (ascii value) for which to search
        pos - optional starting position, defaults to 0
        step - optional step size, defaults to 1
        Returns:
        the index of the first unescaped character, or -1 if character was not found
