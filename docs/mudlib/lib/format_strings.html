      • parse_arg

        public string parse_arg(int token, string *parts, int size, string part, int cursor, string def)
        Parse a format token which takes an optional argument.
        Parameters:
        token - the token (i.e. the 't' part of %t{%d})
        parts - an array of all the tokenized parts of the format string
        size - sizeof(parts)
        part - the part currently being parsed
        cursor - the index of the part currently being parsed, passed by reference (this cursor may need to be advanced)
        def - default value of arg if no arg could be found, or 0 to return 0 in the absense of an arg
        Returns:
        the argument string (i.e. the "%d" part of %t{%d})

      • parse_mobile_format

        public closure parse_mobile_format(string format)
        Parse a format string into a closure that can be invoked later by the logger to format a log line. The closure will take the arguments: 
        'verb: the verb being used to travel
'dir:  for out msg, the exit direction. for in msg, the entrance
direction, or 0 if entrance could not be discerned. always 0 for
teleport messages.
        Parameters:
        format - the format string
        Returns:
        the formatter closure, bound to THISO

      • parse_logger_format

        public closure parse_logger_format(string format)
        Parse a format string into a closure that can be invoked later by the logger to format a log line. The closure will take the arguments: 
        'category: the category of the logger
'priority: the priority of the log event
'message: an optional error message
'caller: an array read from
debug_info(DINFO_TRACE, DIT_CURRENT)
for the invoking stackframe, or 0 if no caller was found
        Parameters:
        format - the format string
        Returns:
        the formatter closure, bound to the factory
