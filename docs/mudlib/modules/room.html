      • query_exits

        public mapping query_exits()
        Return the mapping of all the rooms exists. This is a two-dimensional mapping, with the exit direction (e.g. "north") as the keys. The first value should be the program name of the room to which the player will be moved when using the exit. The second value is a flag designating whether or not the exit should be displayed when the user looks inside a room.
        Returns:
        a mapping of the form ([ direction : destination; hidden ])
