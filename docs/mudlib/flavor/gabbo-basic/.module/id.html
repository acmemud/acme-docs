      • query_primary_id

        public string query_primary_id()
        Returns the primary id of the object. An object may only have one primary id. This id can be used to give hints to the player about how to refer to the object on the command line.
        Returns:
        the primary id, or 0 if the object has no primary id

      • query_secondary_ids

        public string *query_secondary_ids()
        Return the secondary ids of the object. An object may have any number of secondary ids. These ids may be used to identify the object, but should not be used when giving player usage instructions.
        Returns:
        the array of secondary ids

      • set_secondary_ids

        public int set_secondary_ids(string *str)
        Set all secondary ids of the object. Any previously set secondary ids will be overwritten by this list.
        Parameters:
        str - the array of secondary ids
        Returns:
        0 for failure, 1 for success

      • add_secondary_id

        public int add_secondary_id(string str)

        Returns:
        0 for failure, 1 for success
