      • prompt

        protected varargs void prompt(object ob, mixed prompt, mapping context, closure callback, varargs mixed *args)
        Set a user's prompt. This will either be the user's base command prompt, or will be a temporary prompt used to gather one-time input. For single-use prompts, you must pass a callback with optional additonal arguments. If no callback is provided, the user's command prompt will be set. The prompt itself may either be a string or a closure returning a string. A context mapping may also be provided to customized the behavior of the prompt.
        Parameters:
        ob - the interactive object to prompt
        prompt - the prompt string or closure
        context - the prompt context
        callback - an optional callback for single-use prompts
        args - extra args for the callback

      • prompt_input

        public void prompt_input(string input, object ob, mixed prompt, mapping context, closure callback, mixed *args)
        The input handler for input prompts. This is a wrapper for the prompt callback passed to input_prompt(). If the callback returns non-zero, the same prompt may be retried depending on the prompt context.
        Parameters:
        input - the input string
        ob - the interactive object that was prompted
        prompt - the prompt string or closure
        context - the prompt context
        callback - an optional callback for single-use prompts
        args - extra args for the callback

      • prompt_info

        protected mixed *prompt_info(object ob)
        Get information on the prompt stack. The first element of the returned array will contain information on the command prompt; the remaining elements will be any pending input prompts, in order of least recently prompted to most recently prompted. Each element will be a sub-array containing: the prompt string or closure, the prompt context, the callback and callback args. The first element will not containing a callback or extra args.
        Parameters:
        ob - the interactive object that was prompted
        Returns:
        the prompt stack, starting with the command prompt and ending with the most recently set input prompt
