      • send_msg

        protected void send_msg(string message, mapping context, object ob, string topic)
        Send a message with the specified context. The context can hold anything, but at the minimum should declare a "sense" that sensor objects can receive the message using. Without a sense, the message may not be delivered.
        Parameters:
        message - the message to send
        context - the message context
        ob - the target object
        topic - the message topic, optional; if not provided, a suitable default for the current object will be used
        Returns:
        the sent message, or 0 if it could not be sent

      • stdout_msg

        protected void stdout_msg(string message, mapping context, object ob, string topic)
        Send a message over STDOUT. Unlike UNIX where STDOUT is a file handle, in MessageLib and SensorMixin it is an "extra sense." Sensor objects are able to receive messages via extra senses in addition to the standard five, and stdout is one of these senses.
        Parameters:
        message - the message to send
        context - the message context
        ob - the target object
        topic - the message topic, optional; if not provided, a suitable default for the current object will be used
        Returns:
        the sent message, or 0 if it could not be sent
