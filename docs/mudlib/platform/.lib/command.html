        the syntax that was applied

      • parse_args_explode

        public string *parse_args_explode(string arg, int pos, int numargs)
        Parse the argument list out of an argument string using the explode method. Using this method splits the entire string by whitespace and uses each substring as an argument. Arguments that contain whitespace "may be quoted" and the quoted string will be treated as a single argument. If an argument limit is provided, the last argument in the returned argument list will gobble up all additional content in the argument string, even if it contains whitespace.
        Parameters:
        arg - the argument string
        pos - the position in the argument string to begin search, passed by reference and assigned the position of the end of the parsed argument list
        numargs - the expected number of arguments; the resulting argument list may have fewer members, but never more
        Returns:
        the array of command line arguments found in argument string
