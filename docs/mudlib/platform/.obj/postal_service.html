      • send_message

        public void send_message(object target, string topic, string message, mapping context, object sender)
        Send a message to a sensor object. This should be the only use of efun::tell_object() in the lib.
        Parameters:
        target - the object to which the message should be delivered, must be a "sensor"
        topic - the message topic, a way to group common messages together
        message - the message to be delivered, a string
        context - the message context, a mapping
        sender - the object doing the sending, or 0 for anonymous messages
        Returns:
        a Message struct representing the sent message, or 0 if message could not be sent
