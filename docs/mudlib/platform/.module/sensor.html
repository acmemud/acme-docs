Program SensorMixin

  • All Inherited Programs:
    • JSONLib
    • MessageLib
    • ObjectLib

    The basic unit of interaction in the game is a "message," the SensorMixin provides objects the capability to "sense" those messages. Without it, any output or messages an object may receive are simply regarded as noise.
