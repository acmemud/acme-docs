API Specification

This document is the API specification for the gabbo mudlib foundation.

See: Description

Packages 
Package Description
/lib
The programs in the /lib directory are intended to be inherited as utility libraries, organized around common programming patterns or tasks.
/modules
The /modules package contians two types of programs: base objects and mixins.
/obj
The /obj package is for programs which are meant to be instanced, rather than inherited.
/obj/auth
/secure
The /secure directory contains the master and simul_efun objects, which are considered to be outside the exclusive authority of the rest of the mudlib.
